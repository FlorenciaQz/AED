### Definiciones matemáticas

1. Valor absoluto: 

  Sea x ∈ Z,

  |x| = x sí x>=0 ó |x| = -x sí x<0

2. Valor mínimo entre dos valores:

Sean a, b ∈ Z,

  Sí a < b, a es el valor mínimo ya que es el más próximo al 0.

3. Función f3:
